# How to properly use quotation mark

## Single

- `‘` = Left Single Quotation Mark = U+2018
- `’` = Right Single Quotation Mark = U+2019

Example:
> ‘kzl’

## Double

- `“` = Left Double Quotation Mark = U+201C
- `”` = Right Double Quotation Mark = U+201D

Example:
> “cenah”

## Notes

Mine:
> Jangan lupa pake `<CTRL>+<SHIFT>+u`.
