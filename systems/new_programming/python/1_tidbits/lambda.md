# Python Lambda

A lambda function is a small anonymous function.

A lambda function can take any number of arguments, but can only have one expression.

`lambda arguments : expression`

## Examples

`1.`

```python
x = lambda a : a + 10
print(x(5))
```

`2.`

```python
x = lambda a, b : a * b
print(x(5, 6))
```

`3.`

```python
x = lambda a, b, c : a + b + c
print(x(5, 6, 2))
```
